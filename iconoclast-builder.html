Iconoclast (builder): Build an AST hierarchy from an S-expression

Description

This library defines methods on generic functions of the architecture.builder-protol in order to create a hierarchy of Iconoclast ASTs .

Requirements

Depends on architecture.builder-protol, s-expression-syntax, and Iconoclast (AST).
