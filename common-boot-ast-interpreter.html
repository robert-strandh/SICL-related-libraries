Common boot (ast interpreter): Evaluate ASTs using an interpreter

Description

This library takes a finalized Iconoclast AST and evaluates it relative to a first-class global environment.

Requirements

Depends-on Iconoclast (AST), Iconoclast (transformations), Common boot (builder), and Closer MOP.
