Clostrum: First-class global environments

Description

This library provides an implementation of first-class global environments. It provides an improvement of the protocol documented in this paper.

Requirements

Depends on Trucler Uses standard classes and generic functions, so CLOS is required.

Maintainers

Currently maintained by Alex Wood

Possible improvements

Some old LaTeX-based documentation is still present in the repository and should be eliminated. Documentation strings need to be processed in order to create (say) a PDF document, possibly with additional general information about the library.
