Common boot (AST to HIR): Turn an AST into High-level Intermediate Representation

Description

This library turns an Iconoclast AST into High-level Intermediate Representation

Requirements

Depends on Iconoclast (AST), Iconoclast (transformations), Common boot (HIR), and Concrete Syntax Tree.
