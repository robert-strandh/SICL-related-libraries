Clostrophilia: An implementation of the CLOS MOP

Description

This library implements each of the different protocols documented ini the book "The Art of the Metaobject Protocol". It also contains an implementation of fast generic dispatch as published in this paper.

This library is nearly complete.

Requirements

Depends on Ecclesia.

Maintainers

Currently maintained by Robert Strandh.
