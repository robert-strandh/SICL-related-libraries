Futhark: Functions in the Strings dictionary

Description

This library provides implementations for the functions in the Strings dictionary of the Common Lisp standard,

Nearly every function has an extensive documentation string associated with it. Functions with keyword parameters have an associated compiler macro that eliminates keyword parsing when possible.

Requirements

The intrinsic system has no dependencies. The extrinsic system depends on the extrinsic system of Constrictor
